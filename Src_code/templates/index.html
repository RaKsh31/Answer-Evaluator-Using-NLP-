How It Works
  • 1
    Extract — Text is pulled from PDF, image (OCR), DOCX, TXT, or Excel.
  • 2
    Segment — Regex & NLP detect Q1, Q2, Part A/B, (a)/(b) patterns.
  • 3
    Match — Student answers are aligned to the answer key by label, then position.
  • 4
    Evaluate — TF-IDF cosine similarity (or Sentence Transformers) scores each answer.
  • 5
    Store — Marks saved to Excel with one sheet per subject.
